# 🚗 Car Showroom Management System

A simple and efficient **Java-based console application** designed to manage a car showroom. This project applies core **Object-Oriented Programming (OOP)** concepts and handles data persistence using file processing.

## 🌟 Features

### 👤 Manager Portal
* **Secure Login:** Protected by a password.
* **Add Cars:** Input brand, model, year, price, and additional details.
* **Remove Cars:** Delete records from the system easily.
* **Data Persistence:** All changes are automatically saved to a text file.

### 👥 Customer View
* **Browse Inventory:** View a formatted list of all available cars in the showroom.

## 🛠️ Concepts Applied
* **Encapsulation:** Using classes to group car data and behaviors.
* **File I/O:** Reading from and writing to `data.txt` to ensure data isn't lost.
* **ArrayList:** Dynamic storage for car objects.
* **Exception Handling:** Managing potential file errors.

* `Car.java`: The blueprint class for car objects.
